## Methodology

All research follows TEOF principles:

1. **Source tracking** — Every claim links to source
2. **Uncertainty flagged** — Estimates marked as estimates
3. **Human verification** — Research is 🤖 until verified ✅
4. **Reality anchors** — Prefer empirical data over theory
5. **Bounded scope** — Research specific questions, not everything

**Ground in external reality, not self-reference.**

When validating a TEOF claim:
1. Search for external sources (peer-reviewed, long track record)
2. Document the source with link
3. Note whether it validates, contradicts, or is orthogonal
4. Update `external-validation.md` summary table

When a claim has no external validation:
- Mark explicitly as "Novel — needs validation"
- Document what would constitute validation
