## Contribution Process
1. Send BTC to the address above (optional: include a signed message referencing the relevant plan/task).
2. Notify the team via issue, bus message, or PR note so the transaction can be linked to specific work.
3. The steward records the tx under `_report/impact/btc-ledger/` using [`docs/impact/btc-ledger.md`](docs/impact/btc-ledger.md) and stores proofs in `_report/impact/btc-ledger/proof/`.
4. Status/manager reports surface cumulative balance and linked efforts once entries exist.

Strategic capture levers (guard sponsorships, observation bounties, decentralized-node packages) are tracked in [`docs/impact/btc-capture-strategy.md`](docs/impact/btc-capture-strategy.md).
