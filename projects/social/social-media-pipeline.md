## Workflow: AI-Generated Video

```
1. IDEATION (Weekly, 30 min)
   │  Claude/GPT: Generate 5-10 debate topics or concepts
   │  Trend tools: Check what's performing (VidIQ, TrendTok)
   │  Select 2-3 for the week
   ▼
2. SCRIPT GENERATION (Per video, 10 min review)
   │  Claude: Write full dialogue/script
   │  Human: Review, tweak for TEOF alignment
   ▼
3. VOICE GENERATION (Automated)
   │  ElevenLabs: Generate character voices
   │  Multiple speakers for debates
   ▼
4. VIDEO GENERATION (Automated)
   │  HeyGen/D-ID: Avatar videos for speakers
   │  Runway/Pika: B-roll, atmospheric visuals
   │  Midjourney: Thumbnail + scene images
   ▼
5. MUSIC & EDIT (Semi-automated)
   │  Suno/Udio: Generate mood-matched background music
   │  CapCut/Descript: Auto-edit, captions, assembly
   ▼
6. CLIPPING (Automated)
   │  Opus Clip: Long-form → 5-10 shorts
   │  Auto-select viral moments
   ▼
7. DISTRIBUTION (Automated)
   │  Repurpose.io / Publer:
   │  ├── YouTube (long-form + Shorts)
   │  ├── TikTok
   │  ├── Instagram Reels
   │  └── Twitter (clips + thread tie-in)
   ▼
8. ANALYTICS & ITERATION (Weekly, 15 min)
   │  What performed? Why?
   │  Feed back into ideation
```

**Time Investment:** ~2-3 hrs/week for 10-20 pieces of content across all platforms

## Anti-Oneitis Mechanism

Active profile with engagement = abundance mindset. Inbound attention from multiple sources dilutes fixation on any single person.

- YouTube/TikTok: Reach → followers → DMs
- Instagram: Personal brand → dating options
- Twitter: Authority → network connections

**Key insight:** Network creates options. Options prevent oneitis. AI automation makes network-building sustainable alongside job + gym + life.
